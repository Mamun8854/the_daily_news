Differences between var, let, and const

Var : i) The scope of a var variable is functional scope. ii) It can be updated and re-declared into the scope.

Let : i) The scope of a let variable is block scope. ii)It can be updated but cannot be re-declared into the scope.

const : i) The scope of a const variable is block scope. ii) It cannot be updated or re-declared into the scope.
